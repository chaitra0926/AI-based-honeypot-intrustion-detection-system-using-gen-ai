# SentinelAI

## AI Powered Honeypot Intrusion Detection Dashboard

SentinelAI is a portfolio-focused full-stack cybersecurity dashboard that simulates honeypot events,
assigns AI-inspired threat scores and visualizes security incidents through an interactive interface.

## Features

- Dashboard analytics
- Honeypot events table
- Alerts page
- Blocked IP management
- Reports
- Attack knowledge base
- Dummy live event generator
- REST API architecture

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS
- Recharts

Backend:
- FastAPI
- Python

- SQLite
